# 🚗 Electric Vehicle Analysis Dashboard

An interactive Electric Vehicle (EV) Analysis Dashboard built using Power BI to analyze the growth, adoption, and distribution of Electric Vehicles (EVs) across different regions, manufacturers, and vehicle models.

---

# 📌 Project Overview

This dashboard provides detailed insights into the Electric Vehicle ecosystem using interactive visualizations and KPIs.

The project focuses on:

- Total Electric Vehicles
- Average Electric Range
- BEV & PHEV Analysis
- EV Growth Trends
- Geographic Distribution
- CAFV Eligibility
- Top EV Makes & Models

---

# 🎯 KPI Requirements

## 1. Total Vehicles
- Understand the overall EV market size and growth.

## 2. Average Electric Range
- Analyze technological advancement and efficiency of EVs.

## 3. Total BEV Vehicles & % of Total BEV Vehicles
- Identify total Battery Electric Vehicles (BEVs).
- Calculate BEV percentage contribution.

## 4. Total PHEV Vehicles & % of Total PHEV Vehicles
- Identify total Plug-in Hybrid Electric Vehicles (PHEVs).
- Calculate PHEV percentage contribution.
